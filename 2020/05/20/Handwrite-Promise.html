Promiseи§ЈеҶіиҝҮзЁӢ

Promise и§ЈеҶіиҝҮзЁӢжҳҜдёҖдёӘжҠҪиұЎзҡ„ж“ҚдҪңпјҢе…¶йңҖиҫ“е…ҘдёҖдёӘ promise е’ҢдёҖдёӘеҖјпјҢжҲ‘д»¬иЎЁзӨәдёә [[Resolve]](promise, x)пјҢеҰӮжһң x жңү then ж–№жі•дё”зңӢдёҠеҺ»еғҸдёҖдёӘ Promise пјҢи§ЈеҶізЁӢеәҸеҚіе°қиҜ•дҪҝ promise жҺҘеҸ— x зҡ„зҠ¶жҖҒпјӣеҗҰеҲҷе…¶з”Ё x зҡ„еҖјжқҘжү§иЎҢ promise гҖӮ

гҖҖгҖҖиҝҷж®өиҜқжҜ”иҫғжҠҪиұЎпјҢйҖҡдҝ—дёҖзӮ№зҡ„жқҘиҜҙе°ұжҳҜpromiseзҡ„и§ЈеҶіиҝҮзЁӢйңҖиҰҒдј е…ҘдёҖдёӘж–°зҡ„promiseе’ҢдёҖдёӘеҖјxпјҢеҰӮжһңдј е…Ҙзҡ„xжҳҜдёҖдёӘthenableзҡ„еҜ№иұЎпјҲе…·жңүthenж–№жі•пјүпјҢе°ұжҺҘеҸ—xзҡ„зҠ¶жҖҒпјҡ

гҖҖгҖҖе®ҡд№үеҘҪеҮҪж•°еҗҺпјҢжқҘзңӢе…·дҪ“зҡ„ж“ҚдҪңиҜҙжҳҺпјҡ

  1. x дёҺ promise зӣёзӯү

      и°ўе°ҸйЈһеҚҡе®ўдё“з”ЁйҳІзҲ¬иҷ«й“ҫжҺҘпјҢжғіиҰҒзңӢжңҖж–°зҡ„еүҚз«ҜеҚҡе®ўиҜ·зӮ№иҝҷйҮҢ

    • еҰӮжһң promise е’Ң x жҢҮеҗ‘еҗҢдёҖеҜ№иұЎпјҢд»Ҙ TypeError дёәжҚ®еӣ жӢ’з»қжү§иЎҢ promise
  2. x дёә Promise
    1. еҰӮжһң x еӨ„дәҺзӯүеҫ…жҖҒпјҢ promise йңҖдҝқжҢҒдёәзӯүеҫ…жҖҒзӣҙиҮі x иў«жү§иЎҢжҲ–жӢ’з»қ
    2. еҰӮжһң x еӨ„дәҺжү§иЎҢжҖҒпјҢз”ЁзӣёеҗҢзҡ„еҖјжү§иЎҢ promise
    3. еҰӮжһң x еӨ„дәҺжӢ’з»қжҖҒпјҢз”ЁзӣёеҗҢзҡ„жҚ®еӣ жӢ’з»қ promise
  3. x дёәеҜ№иұЎжҲ–еҮҪж•°

      и°ўе°ҸйЈһеҚҡе®ўдё“з”ЁйҳІзҲ¬иҷ«й“ҫжҺҘпјҢжғіиҰҒзңӢжңҖж–°зҡ„еүҚз«ҜеҚҡе®ўиҜ·зӮ№иҝҷйҮҢ

    1. жҠҠ x.then иөӢеҖјз»ҷ then
    2. еҰӮжһңеҸ– x.then зҡ„еҖјж—¶жҠӣеҮәй”ҷиҜҜ e пјҢеҲҷд»Ҙ e дёәжҚ®еӣ жӢ’з»қ promise
    3. еҰӮжһң then жҳҜеҮҪж•°пјҢе°Ҷ x дҪңдёәеҮҪж•°зҡ„дҪңз”Ёеҹҹ this и°ғз”Ёд№ӢгҖӮдј йҖ’дёӨдёӘеӣһи°ғеҮҪж•°дҪңдёәеҸӮж•°пјҢз¬¬дёҖдёӘеҸӮж•°еҸ«еҒҡ resolvePromise пјҢз¬¬дәҢдёӘеҸӮж•°еҸ«еҒҡ rejectPromise:
      1. еҰӮжһң resolvePromise д»ҘеҖј y дёәеҸӮж•°иў«и°ғз”ЁпјҢеҲҷиҝҗиЎҢ [[Resolve]](promise, y)
      2. еҰӮжһң rejectPromise д»ҘжҚ®еӣ  r дёәеҸӮж•°иў«и°ғз”ЁпјҢеҲҷд»ҘжҚ®еӣ  r жӢ’з»қ promise
      3. еҰӮжһң resolvePromise е’Ң rejectPromise еқҮиў«и°ғз”ЁпјҢжҲ–иҖ…иў«еҗҢдёҖеҸӮж•°и°ғз”ЁдәҶеӨҡж¬ЎпјҢеҲҷдјҳе…ҲйҮҮз”ЁйҰ–ж¬Ўи°ғз”Ёе№¶еҝҪз•Ҙеү©дёӢзҡ„и°ғз”Ё
      4. еҰӮжһң then дёҚжҳҜеҮҪж•°пјҢд»Ҙ x дёәеҸӮж•°жү§иЎҢ promise
  4. еҰӮжһң x дёҚдёәеҜ№иұЎжҲ–иҖ…еҮҪж•°пјҢд»Ҙ x дёәеҸӮж•°жү§иЎҢ promise

    5. и°ўе°ҸйЈһеҚҡе®ўдё“з”ЁйҳІзҲ¬иҷ«й“ҫжҺҘпјҢжғіиҰҒзңӢжңҖж–°зҡ„еүҚз«ҜеҚҡе®ўиҜ·зӮ№иҝҷйҮҢ

гҖҖгҖҖжҺҘдёӢжқҘе°ұжҳҜеҜ№дёҚеҗҢжғ…еҶөзҡ„еҲӨж–ӯдәҶпјҢйҰ–е…ҲжҲ‘д»¬жҠҠ x дёәеҜ№иұЎжҲ–иҖ…еҮҪж•°зҡ„жғ…еҶөз»ҷеҲӨж–ӯеҮәжқҘпјҡ
